About FFXIV Tools

What is this?

FFXIV Tools is a collection of utilities designed to help Final Fantasy XIV players with various in-game calculations and tracking needs.

Available Tools

  • FC Calculator: Calculate the exact number of items needed to reach Free Company Rank 6, with support for different item levels and multiple FCs.
  • Quest Tracker: Compare Questionable's current quest paths against the FFXIV quest database to identify missing coverage.
  • Aether Currents: Interactive maps showing locations of Aether Currents and Aetherytes across different zones.

Features

  • Real-time calculations based on official game formulas
  • Interactive visual maps with hover tooltips
  • Responsive design that works on desktop and mobile
